# Clockd legal pages (public)

Public GitHub Pages for the Clockd iOS app — privacy policy and support URL for App Store Connect.

| Page | URL |
|------|-----|
| Support | https://garrett149b123.github.io/clockd-legal/ |
| Privacy | https://garrett149b123.github.io/clockd-legal/privacy-policy.html |

The app source code lives in the private [clockd](https://github.com/garrett149b123/clockd) repo.

## GitHub Pages setup (one time)

1. Repo **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder **/ (root)**
4. Save and wait ~1 minute

## Update policy text

Edit `privacy-policy.html` or `index.html` in this repo, commit, and push to `main`.
